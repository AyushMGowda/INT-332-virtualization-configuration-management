# Docker Image Management

### Docker Hub
Docker Hub is a cloud-based registry service provided by Docker, Inc. used to store, share, and distribute Docker container images.
Docker Hub hosts verified images maintained by Docker or trusted partners.
These are some of the Images which can be found on Docker Hub.

## Hello-World
The hello-world image is the simplest Docker test container
```bash
docker pull hello-world
```
## Ubuntu
Ubuntu is a lightweight Linux distribution commonly used as a base image for containers and development environments.
```bash
docker pull ubuntu
```
## NGINX
Nginx is a high-performance web server and reverse proxy used for serving web applications and load balancing.
```bash
docker pull nginx
```
## MySQL
MySQL is an open-source relational database management system used for storing structured data.
```bash
docker pull mysql
```
## Postgres
PostgreSQL is a powerful open-source relational database known for reliability, scalability, and advanced SQL features.
```bash
docker pull postgres
```
## MongoDB
MongoDB is a NoSQL database that stores data in flexible JSON-like documents.
```bash
docker pull mongo
```
## Redis
Redis is an in-memory key-value data store used for caching, session storage, and real-time analytics.
```bash
docker pull redis
```
## Python
Python (programming language) image provides a ready environment to run Python applications and scripts inside containers.
```bash
docker pull python
```
## NodeJS
Node.js is a JavaScript runtime used for building scalable backend applications and APIs.
```bash
docker pull node
```
## OpenJDK
OpenJDK is the open-source implementation of the Java Development Kit used to run and build Java applications.
```bash
docker pull openjdk
```
## Tomcat
Apache Tomcat is a web server and servlet container used to deploy Java web applications.
```bash
docker pull tomcat
```
## PHP
PHP is a server-side scripting language commonly used for building dynamic web applications.
```bash
docker pull php
```
## Golang
Go (programming language) provides an environment to build and run applications written in the Go programming language.
```bash
docker pull golang
```
